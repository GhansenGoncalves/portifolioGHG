## ⭐ Projeto em destaque

### [🍰 Bolos da Bru — sistema de vendas de bolo no pote](bolos-da-bru/)

Sistema web completo desenvolvido e testado de ponta a ponta: dashboard com
indicadores e gráficos, vendas de balcão/delivery/encomendas, loja on-line para
o cliente, promoções automáticas e dicas geradas dos dados — com **26 testes
E2E em Playwright rodando em CI** a cada push.

[![Dashboard do Bolos da Bru](bolos-da-bru/docs/screenshots/dashboard.png)](bolos-da-bru/)

**Stack:** HTML, CSS e JavaScript puros (sem dependências de runtime) ·
Playwright · GitHub Actions
**Destaques de QA:** testes isolados e determinísticos, cobertura dos fluxos
críticos de negócio (estoque, promoções, encomendas, cancelamento com estorno),
validação de acessibilidade (paleta segura para daltonismo, navegação por
teclado, tabela alternativa para cada gráfico).

→ [Ver o projeto, screenshots e como rodar](bolos-da-bru/README.md)
